# To Do Application
Created by Team SKEMA 

## API for To Do App Documentation:
```node /Routes/todos.js```

## To Do Endpoints


### Get a List of All Todos
``` GET /todos/ ```



### Add a To Do 
``` POST /todos/ ```



### Edit/Update a To Do 
``` PUT /todos/:id ```
|      :id        |  type  |
|-----------------|--------|
|        4        | number |


### Delete a To Do
``` DELETE /todos/:id ```

**Path Parameters**
|      :id        |  type  |
|-----------------|--------|
|        4        | number |
